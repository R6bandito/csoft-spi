# csoft-spi – 轻量级软件 SPI 库（含延时工具）

**csoft-spi** 是一个纯 C 语言实现的软件 SPI 协议栈，专为资源受限或需要灵活控制 SPI 时序的嵌入式系统设计。  
它不依赖任何硬件 SPI 外设，只需普通 GPIO 即可模拟 SPI 总线，支持 **4 种 SPI 模式**、**MSB/LSB 位序切换** 和 **多设备总线管理**。

**csoft-spi** is a lightweight software SPI library written in pure C. It is designed for embedded systems with limited hardware SPI resources or those requiring flexible SPI timing control. No hardware SPI peripheral is needed – only standard GPIOs. Features include **all 4 SPI modes**, **MSB/LSB first ordering**, and **multi‑device bus management**.
